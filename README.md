# 교과 수행평가 세특 메이커

한국 고등학교의 여러 교과에서 수행평가와 학생 관찰 자료를 정리하고, 세특 작성용 AI 프롬프트와 오프라인 보조 초안을 만드는 정적 웹앱 프로젝트입니다.

## 현재 단계

이 프로젝트는 안정판인 **수학 수행평가 세특 메이커**를 복사해 만든 다교과 확장 개발판입니다. 기존 개별·학급 작성, 엑셀, 백업 기능은 보존되어 있지만 현재 교과 데이터는 아직 수학 중심입니다.

## 개발 방향

- 과목별 성취기준
- 과목별 핵심 역량과 행동 기준
- 과목별 세특 작성 규칙
- 개별 및 학급 일괄 작성
- 엑셀 입력·출력
- 전체 데이터 백업·복구
- 수학 안정판과 독립된 브라우저 저장 공간
